Link de download do Git: https://git-scm.com/downloads
O Git Bash é um terminal extendido para otimizar o uso do Git

Comandos:
cd : change directory - muda de pasta
cd .. : Volta a pasta anterior
mkdir : make directory - cria uma nova pasta
rm -rf : remove pasta, flag -rf serve para remover a paste e conteudos sem predir permissão
